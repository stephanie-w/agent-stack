# Shape Spec

Use the `shape-spec` skill to gather context and structure the technical plan for a feature.

1. Verify that you are currently in "Plan Mode" (if not, instruct the user to enter plan mode and stop).
2. LOAD the `shape-spec` skill.
3. Execute the shaping workflow to create the specification documents.
